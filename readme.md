# Projet SnP500

## Explication du projet

### Prérequis : 
- Avoir python installé sur son pc
- Activer un venv avant de faire quoi que ce soit

### Quel est son but ? 
- Donner le prix du SnP500 
- Apprendre les bases de python
- Apprendre à utiliser les outils comme : ClaudeCode, Git/GitHub, lib etc...

### Comment l'installer et l'utiliser l'app Python ?
- Aller sur le GitHub d'@noecuccia ou sur ce lien : https://github.com/noecuccia/SnP500v1
- Cloner le projet "app.py" 
- L'ouvrir sur une console 
- Puis éxecuter : py app.py

### Commnent installer et utiliser l'app Web Python ?
- Aller sur le GitHub d'@noecuccia ou sur ce lien : https://github.com/noecuccia/SnP500v1
- Cloner le projet "web_app.py"
- L'ouvrir sur une console 
- Puis éxecuter : py web_app.py
- Copier le lien et le mettre sur un navigateur internet
